# 💼 Portfolio Dev — Praticando HTML e CSS

Este projeto foi desenvolvido como parte dos estudos com a **Rocketseat**, com o objetivo de praticar a construção de páginas estáticas utilizando **HTML e CSS**, reforçando os fundamentos do front-end na prática.

---

## 📌 Objetivo

> Fixar conceitos essenciais de **estruturação com HTML** e **estilização com CSS**, aplicando boas práticas de organização de layout, tipografia e composição visual em um portfólio profissional.

Apesar de simples, esse projeto é excelente para:

* Consolidar a base de construção de páginas web
* Criar uma estrutura de portfólio pessoal
* Trabalhar com organização de conteúdo visual
* Praticar espaçamentos, cores e hierarquia tipográfica
* Desenvolver uma melhor percepção de layout e design

---

## 🧠 O que aprendi com este projeto

* Como montar uma **estrutura semântica** em HTML
* Organização de seções como apresentação, projetos e contato
* Aplicação de **CSS para estilização e layout**
* Uso de **variáveis CSS (:root)** para padronização
* Melhoria na organização e legibilidade do código
* Construção de layouts mais bem estruturados

---

## 🛠️ Tecnologias Utilizadas

* **HTML5** – estrutura da página com marcação semântica
* **CSS3** – estilização com foco em layout, cores e tipografia
* **CSS Variables** – padronização de estilos

---

## 🖥️ Como rodar o projeto localmente

1. Clone o repositório:

```bash
git clone https://github.com/wemersonmatheuss/rocketseat-projects.git
```

2. Acesse a pasta do projeto:

   ```bash
   cd rocketseat-projects/html-css/portfoliodev-html-css
   ```

3. Abra o arquivo index.html no navegador
   3.1 Clique duas vezes no arquivo `index.html`, ou utilize a extensão *Live Server* no VS Code
