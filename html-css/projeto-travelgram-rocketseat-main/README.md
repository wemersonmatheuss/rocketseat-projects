# ✈️ Travelgram — Praticando responsividade e fundamentos de CSS

Este projeto foi desenvolvido como exercício proposto pela **Rocketseat**, com o objetivo de treinar **responsividade** e reforçar fundamentos importantes do **CSS** na prática.

---

## 📌 Objetivo

> Fixar conceitos essenciais de **estruturação com HTML** e **estilização com CSS**, explorando boas práticas de layout responsivo, uso de flexbox e organização de componentes visuais.

Apesar de simples, esse projeto é excelente para:
- Consolidar a lógica de construção visual de uma página
- Aprender a lidar com espaçamentos e alinhamentos
- Praticar o uso de media queries
- Desenvolver uma visão mais refinada de design para múltiplos dispositivos

---

## 🧠 O que aprendi com este projeto

- Como montar uma **estrutura semântica** em HTML
- Utilização de **Flexbox** para posicionar elementos
- Aplicação de **media queries** para tornar a página responsiva
- A importância de organizar bem as **classes CSS**
- Melhoria na leitura e organização do código
- Como pensar a responsividade desde o início do desenvolvimento

---

## 🛠️ Tecnologias Utilizadas

- **HTML5** – estrutura da página com marcação semântica
- **CSS3** – estilização com foco em layout, cores, tipografia e responsividade
- **Responsividade** com media queries e Flexbox
