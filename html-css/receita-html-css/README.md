# 🍽️ Recipe Page — Praticando HTML e CSS

Este projeto foi desenvolvido como exercício durante os estudos com a **Rocketseat**, com o objetivo de praticar a construção de páginas estáticas utilizando **HTML e CSS**, reforçando os fundamentos do front-end na prática.

---

## 📌 Objetivo

> Fixar conceitos essenciais de **estruturação com HTML** e **estilização com CSS**, aplicando boas práticas de organização de layout, tipografia e reutilização de estilos.

Apesar de simples, esse projeto é excelente para:

* Consolidar a base de construção de páginas web
* Aprender a estruturar conteúdos de forma semântica
* Trabalhar com espaçamentos, cores e tipografia
* Desenvolver uma organização mais limpa e reutilizável no CSS

---

## 🧠 O que aprendi com este projeto

* Como montar uma **estrutura semântica** em HTML
* Uso correto de **listas (ul/li)** para ingredientes
* Aplicação de **CSS para layout e estilização**
* Utilização de **variáveis CSS (:root)** para padronização
* Organização de código e separação de responsabilidades
* Melhoria na legibilidade e estrutura do projeto

---

## 🛠️ Tecnologias Utilizadas

* **HTML5** – estrutura da página com marcação semântica
* **CSS3** – estilização com foco em layout, cores e tipografia
* **CSS Variables** – padronização de cores e estilos

---

## 🖥️ Como rodar o projeto localmente

1. Clone o repositório:

   ```bash
   git clone https://github.com/wemersonmatheuss/rocketseat-projects.git
   ```

2. Acesse a pasta do projeto:

   ```bash
   cd rocketseat-projects/html-css/receita-html-css
   ```

3. Abra o arquivo index.html no navegador
   3.1 Clique duas vezes no arquivo `index.html`, ou utilize a extensão *Live Server* no VS Code
