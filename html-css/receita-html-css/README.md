# 🍽️ Recipe Page — Praticando HTML e CSS

Este projeto foi desenvolvido como parte dos estudos da **Rocketseat**, com foco em praticar a construção de páginas estáticas utilizando **HTML e CSS**, aplicando boas práticas de estrutura e estilização.

---

## 📌 Objetivo

> Criar uma página de receita organizada, visualmente agradável e bem estruturada, reforçando conceitos fundamentais do desenvolvimento front-end.

Este projeto ajudou a:

* Trabalhar com **estrutura semântica**
* Aplicar **estilização com CSS**
* Organizar melhor o código
* Criar layouts simples e bem apresentados

---

## 🧠 O que aprendi com este projeto

* Uso correto de **tags semânticas** (h1, h2, p, ul, li)
* Estruturação de conteúdo em uma página real
* Aplicação de **CSS para layout e tipografia**
* Uso de **variáveis CSS (:root)**
* Organização de arquivos e estilos
* Separação entre estrutura (HTML) e estilo (CSS)

---

## 🛠️ Tecnologias Utilizadas

* **HTML5** – estrutura da página
* **CSS3** – estilização e organização visual
* **CSS Variables** – padronização de cores e tipografia
