# 🛒 Lista de Compras — Praticando JavaScript

Este projeto foi desenvolvido como parte dos estudos com a **Rocketseat**, com o objetivo de praticar os fundamentos do **JavaScript**, focando em lógica de programação, manipulação do DOM e interatividade em páginas web.

---

## 📌 Objetivo

> Fixar conceitos essenciais de **JavaScript**, aplicando manipulação de elementos, eventos e controle de dados em uma aplicação prática.

Apesar de simples, esse projeto é excelente para:

* Consolidar a base da lógica de programação
* Trabalhar com manipulação do DOM
* Praticar eventos e interações com o usuário
* Controlar listas dinâmicas (adicionar e remover itens)
* Desenvolver raciocínio lógico aplicado ao front-end

---

## 🧠 O que aprendi com este projeto

* Como adicionar e remover elementos dinamicamente na página
* Manipulação de listas com JavaScript
* Uso de **eventos (click, submit, etc.)**
* Controle de estado simples da aplicação
* Atualização dinâmica da interface
* Melhor organização do código JavaScript

---

## 🛠️ Tecnologias Utilizadas

* **HTML5** – estrutura da aplicação
* **CSS3** – estilização da interface
* **JavaScript** – lógica, interatividade e manipulação do DOM

---

## 🖥️ Como rodar o projeto localmente

1. Clone o repositório:

   ```bash
   git clone https://github.com/wemersonmatheuss/rocketseat-projects.git
   ```

2. Acesse a pasta do projeto:

   ```bash
   cd rocketseat-projects/javascript/shopping
   ```

3. Abra o arquivo index.html no navegador
   3.1 Clique duas vezes no arquivo `index.html`, ou utilize a extensão *Live Server* no VS Code
